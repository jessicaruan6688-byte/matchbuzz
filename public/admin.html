How it works

Provider behavior

  • If `GMI_API_*` is configured, content generation uses the real GMI Cloud endpoint first
  • If `MATCH_DATA_API_*` is configured, the match list uses the remote feed first
  • If either provider is unavailable, the app falls back to the local workflow-safe mode
  • Campaign counts show whether the operator workflow is only rendered or actually being used
